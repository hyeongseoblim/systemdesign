NAT Gateway 비용 함정
NAT Gateway를 통한 S3, DynamoDB 트래픽은 GB당 요금이 발생한다. VPC Endpoint를 활용하면 NAT를 거치지 않고 AWS 내부망으로 통신 가능 — 비용 절감 + 보안 강화 동시 달성.

"Multi-AZ 구성은 왜 하나요?" — 단일 AZ 장애(IDC 전원, 네트워크 장애) 시 서비스 중단 방지. RDS Multi-AZ는 동기 복제로 Failover 30초~2분. ALB는 자동으로 헬시한 AZ로 라우팅. RTO(Recovery Time Objective, 복구 목표 시간)와 RPO(Recovery Point Objective, 복구 목표 시점) 관점에서 비용 대비 효과를 설명해야 한다.

Kubernetes 함정
  • Liveness probe와 Readiness probe 혼동: Liveness는 재시작 조건, Readiness는 트래픽 수신 조건. 무거운 작업 중 Liveness 실패 → 재시작 루프

  • latest 태그 사용 금지: 이미지 버전 불명확, 롤백 불가

4. CI/CD 파이프라인

CI(Continuous Integration, 지속적 통합)는 코드 변경마다 자동 빌드/테스트. CD(Continuous Delivery/Deployment, 지속적 배포)는 스테이징/운영 자동 배포까지 확장.

GitOps

GitOps는 Git을 단일 진실 원천(Single Source of Truth)으로 삼아 인프라 상태를 선언적으로 관리하는 방식이다. ArgoCD(아르고시디)가 Git 저장소를 감시하다가 변경 감지 시 클러스터에 자동 동기화(Pull 기반).
